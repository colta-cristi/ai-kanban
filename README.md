# Kanban Project Manager

A simple Kanban board MVP built with Next.js.

## Run

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Test

```bash
cd frontend
npm test          # unit tests (Vitest)
npm run test:e2e  # end-to-end tests (Playwright)
```

## Build

```bash
cd frontend
npm run build
npm start
```
